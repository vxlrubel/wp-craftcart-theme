How to use Wishlist

  • 01. Install the plugin called "YITH WooCommerce Wishlist"
  • 02. Activate the plugin
  • 03. Goto setting of "YITH WooCommerce Wishlist" plugin. settings
  • 04. Collapse the settings section and click the "Add to wishlist".
  • 05. Select (Show "Add to wishlist" button ) from general setting.
  • 06. Use short code [yith_wcwl_add_to_wishlist] from Loop settings.
  • 07. Use short code [yith_wcwl_add_to_wishlist] from Product page settings.
  • 08. All input value will be empty from Text customization.
  • 09. Select (Button with theme style ) from Style & Color customization.
  • 10. Click "Save Options" button and enjoy.
